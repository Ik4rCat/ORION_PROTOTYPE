# Инструкция по настройке новых компонентов

## 1. Настройка кнопки "Продолжить" на панели регистрации

1. Откройте сцену с формой регистрации
2. В объекте RegisterPanel добавьте новую кнопку:
   - Правый клик → UI → Button
   - Переименуйте в "ContinueButton"
   - Настройте внешний вид:
     * Text: "Продолжить"
     * Width: 200, Height: 50
     * Position: поместите под кнопкой регистрации
     * Normal Color: светло-зеленый (#2ECC71)
     * Transition: Color Tint
   - Скройте кнопку, сняв флажок активности в инспекторе

3. Настройте компонент RegisterForm:
   - Выберите панель RegisterPanel
   - В инспекторе найдите компонент RegisterForm
   - В поле Continue Button перетащите созданную кнопку ContinueButton

## 2. Настройка тестирования сохранения данных

1. Добавьте скрипт DashboardPanel.cs на объект DashboardPanel:
   - Выберите панель DashboardPanel
   - Add Component → Scripts → Tests → DashboardPanel

2. Создайте кнопку для тестирования:
   - В DashboardPanel: правый клик → UI → Button
   - Переименуйте в "TestSaveButton"
   - Настройте внешний вид:
     * Text: "Проверить сохранение данных"
     * Width: 300, Height: 50
     * Position: разместите удобно на панели
     * Normal Color: синий (#3498DB)

3. Создайте текстовое поле для результатов:
   - В DashboardPanel: правый клик → UI → Text - TextMeshPro
   - Переименуйте в "TestResultText"
   - Настройте:
     * Width: 600, Height: 300
     * Position: под кнопкой TestSaveButton
     * Font Size: 18
     * Пустой текст изначально
     * Alignment: Left, Top
     * Text Wrapping: включить
     * Color: белый (#FFFFFF)

4. Настройте компонент DashboardPanel:
   - В инспекторе найдите компонент DashboardPanel
   - В поле Test Save Button перетащите кнопку TestSaveButton
   - В поле Test Result Text перетащите текстовое поле TestResultText
   - В поле User Info Text перетащите текстовое поле для отображения информации о пользователе

5. Настройте компонент LoginUIManager:
   - Выберите объект AuthSystem
   - В инспекторе найдите компонент LoginUIManager
   - В поле Dashboard Panel Component перетащите DashboardPanel

## 3. Проверка функциональности

1. Запустите сцену в режиме Play
2. Создайте нового пользователя:
   - На форме регистрации введите данные
   - После успешной регистрации вы должны автоматически попасть на панель Dashboard
   - Информация о пользователе должна отображаться
   
3. Проверьте сохранение данных:
   - Нажмите кнопку "Проверить сохранение данных"
   - В текстовом поле должны отобразиться результаты проверки
   - Убедитесь, что файлы создаются в указанной директории

4. Выйдите и войдите снова:
   - Нажмите кнопку "Выход"
   - Введите данные пользователя на форме входа
   - Убедитесь, что вход выполняется успешно

## 4. Отладка проблем

Если возникают проблемы:

1. Проверьте консоль Unity на наличие ошибок
2. Убедитесь, что пути к файлам сохранения правильно настроены
3. В режиме отладки выведите на консоль значение `Application.persistentDataPath` и проверьте наличие файлов
4. Убедитесь, что все ссылки на компоненты правильно установлены в инспекторе
